===============================================================
Personalizza il monitoraggio della qualità dei servizi offerti 
===============================================================

TVox Omnichannel Contact Center fornisce una reportistica standard ed un set di dashboard real-time che consentono un monitoraggio dei livelli di servizio e di occupazione delle risorse del Contact Center.

Nel momento in cui si ha necessità di creare report e dashboard personalizzati, ad esempio per integrare i dati del TVox con quelli del tuo CRM, Telenia mette a disposizione una Web API per l'accesso diretto al dato reportizzato dei canali abilitati: **TVox Data Model**.
